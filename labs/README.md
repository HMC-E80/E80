# E80 Lab Exercises

This directory contains all lab exercises for the E80 Experimental Engineering course.

## Directory Structure

- `lab01` - Introduction to Arduino programming and basic motor control
- `lab05` - Teensy-based data acquisition system
- `lab07_dive` - Dive controller for underwater robot
- `lab07_surface` - Surface controller for underwater robot

## Setup Instructions

Each lab directory contains an Arduino sketch (.ino file) that can be opened with the Arduino IDE. Make sure to install the Teensyduino add-on and required libraries listed in the main project README.
